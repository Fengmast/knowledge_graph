# 高中物理选择性必修2知识图谱

## 📋 项目简介

本项目为2019年版高中物理选择性必修2教材创建了五个版本的知识图谱，包括：
1. **新人教版**选择性必修二
2. **沪科技版**选择性必修二  
3. **教科版**选择性必修二
4. **粤教版**选择性必修二
5. **鲁教版**选择性必修二

## 📂 文件说明

### 主要文件
- `index.html` - **总览页面**（从这里开始浏览）
- `知识图谱_新人教版选择性必修二.html` - 新人教版知识图谱
- `知识图谱_沪科技版选择性必修二.html` - 沪科技版知识图谱
- `知识图谱_教科版选择性必修二.html` - 教科版知识图谱
- `知识图谱_粤教版选择性必修二.html` - 粤教版知识图谱
- `知识图谱_鲁教版选择性必修二.html` - 鲁教版知识图谱

### 数据文件
- `知识图谱数据(1).docx` - 原始教材数据
- `textbook_data.json` - 提取的JSON格式数据

### 脚本文件
- `create_knowledge_graph.py` - 知识图谱生成脚本
- `parse_and_visualize.py` - 数据解析脚本
- `extract_docx.py` - 文档提取脚本

## 🚀 使用方法

### 查看知识图谱
1. **双击打开 `index.html`** 查看总览页面
2. 点击任意卡片进入对应版本的知识图谱
3. 在图谱页面中：
   - **拖动节点**：调整布局
   - **滚轮缩放**：放大/缩小视图
   - **点击节点**：查看详细信息
   - **拖动背景**：移动整个图谱

### 重新生成图谱
如果需要修改数据并重新生成图谱：
```bash
python create_knowledge_graph.py
```

## 🎨 知识图谱层次结构

知识图谱采用四级层次结构：

```
📕 教材（红色，最大节点）
  └─ 📖 章（青色，大节点）
      └─ 📝 节（绿色，中等节点）
          └─ 💡 知识点（粉色，小节点）
```

### 颜色说明
- 🔴 **红色** - 教材名称（中心节点）
- 🔵 **青色** - 章节
- 🟢 **绿色** - 小节  
- 🔴 **粉色** - 具体知识点

## 📊 教材内容概览

### 共同主题
所有五个版本都包含以下核心内容：
- **磁场与磁力**：安培力、洛伦兹力
- **电磁感应**：法拉第电磁感应定律、楞次定律
- **交变电流**：交流电的产生、变压器、远距离输电
- **电磁波**：麦克斯韦理论、电磁波谱
- **传感器**：各类传感器原理与应用

### 各版本特点
- **新人教版**：55个知识点，结构清晰，注重基础
- **沪科技版**：55个知识点，强调科技应用
- **教科版**：67个知识点，内容最丰富
- **粤教版**：60个知识点，注重实验探究
- **鲁教版**：42个知识点，内容精炼

- **pyvis** - 交互式网络图可视化

- **HTML/CSS/JavaScript** - 前端展示
