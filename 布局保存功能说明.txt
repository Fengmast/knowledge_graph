💾 知识图谱布局保存功能详细说明
==================================================

✅ 功能介绍
--------------------------------------------------
现在知识图谱支持保存您调整后的节点位置！
调整一次，永久保存，下次打开自动恢复！

==================================================

📝 使用步骤
--------------------------------------------------

第1步：打开知识图谱
   → 双击任意知识图谱HTML文件
   → 在浏览器中打开

第2步：调整节点位置
   → 用鼠标拖动任意节点到理想位置
   → 可以拖动多个节点
   → 可以缩放视图查看不同区域
   → 在空白处拖动可移动整个视图

第3步：保存布局 ⭐重要⭐
   → 调整满意后，点击右上角「💾 保存当前布局」按钮
   → 按钮会变成绿色，显示「✓ 已保存！」
   → 2秒后恢复原样
   → 此时布局已保存成功！

第4步：验证保存
   → 关闭浏览器页面
   → 重新打开同一个知识图谱文件
   → 您会发现节点位置完全保持您保存时的状态
   → 自动加载，无需任何操作！

==================================================

🔧 其他功能
--------------------------------------------------

🔄 恢复初始布局
   如果您想重新开始，点击「🔄 恢复初始布局」按钮
   → 会清除您保存的位置
   → 页面会刷新，回到最初的布局
   → 然后您可以重新调整并保存

💡 多次调整
   → 您可以随时调整节点位置
   → 每次调整后都可以点击保存
   → 新的保存会覆盖之前的保存
   → 始终保存的是最后一次点击保存时的状态

==================================================

❓ 常见问题
--------------------------------------------------

Q1：保存的布局在哪里？
A1：保存在浏览器的localStorage中（本地存储）
    不会上传到网络，完全在您的电脑本地

Q2：换一台电脑打开还能看到保存的布局吗？
A2：不能。布局保存在当前浏览器的本地存储中
    换电脑或换浏览器会显示初始布局
    需要在新电脑/浏览器上重新调整和保存

Q3：清除浏览器缓存会丢失保存的布局吗？
A3：会。如果清除浏览器的localStorage数据，保存的布局会丢失
    但只要不清除缓存，布局会永久保存

Q4：五个版本的知识图谱是分别保存的吗？
A4：是的！每个版本的布局独立保存
    互不影响，可以分别调整

Q5：我没点保存就关闭了页面，调整会丢失吗？
A5：会。必须点击「💾 保存当前布局」按钮才能保存
    否则调整会在关闭页面后丢失

💡 最佳实践
--------------------------------------------------

1. 分区域调整
   → 先调整章节层级的布局
   → 再调整小节层级
   → 最后微调知识点位置
   → 每个阶段都可以保存一次

2. 利用缩放功能
   → 缩小视图看整体布局
   → 放大视图调整局部细节
   → 移动视图切换不同区域

3. 定期保存
   → 调整一部分后就保存一次
   → 避免长时间调整后忘记保存
   → 保存是瞬时的，不用担心频繁保存

4. 对比不同版本
   → 可以把五个版本都调整成相似的结构
   → 方便对比不同教材的知识体系差异

🎯 技术说明
--------------------------------------------------

保存原理：
- 使用浏览器的localStorage API
- 保存每个节点的x、y坐标
- 以JSON格式存储
- 页面加载时自动读取并应用

存储位置：
- Chrome: C:\Users\[用户名]\AppData\Local\Google\Chrome\...
- Edge: C:\Users\[用户名]\AppData\Local\Microsoft\Edge\...
- 具体路径由浏览器管理，用户无需关心

存储容量：
- 每个域名约5-10MB的localStorage空间
- 一个知识图谱的位置数据仅几KB
- 完全不用担心空间问题
